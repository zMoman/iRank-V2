Dynamic Playlist Management Web Application

Developed by: Mohammed Al Amri

Overview

This web application allows users to create, manage, and interact with their custom music playlists. Built using JavaScript, Node.js, Express.js, HTML, CSS, and SQLite, the app integrates with the iTunes API to enable song searches and playlist functionality.

Features

Search and add songs dynamically from the iTunes API
Create, reorder, and manage custom playlists
User authentication and registration system using SQLite
Persistent playlist data via local storage
Backend server built with Node.js and Express
Installation Instructions

Follow these steps to set up the project locally:

Clone the repository:
git clone https://github.com/zMoman/iRank-V2.git

Navigate to the project directory:
cd iRank-V2

Install project dependencies:
npm install

Launch Instructions

To run the application locally:

Start the server:
npm start

Open your browser and navigate to:
Main App: http://localhost:5000/

Testing Instructions
You can test the app by running it on your local machine and interacting with the following routes:
